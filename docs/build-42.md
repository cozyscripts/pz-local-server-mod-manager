# Build 42 Setup

Use the same Project Zomboid branch as the mods you install. If your Workshop items are marked for Build 42, set Project Zomboid in Steam to the Build 42 branch before hosting.

The dedicated server helper installs Steam app `380870` and defaults to the `unstable` branch. If The Indie Stone changes the branch name later, update it in Settings before running **Update Dedicated Tool**.

Host mode remains the primary workflow: open Project Zomboid, choose **Host**, select the profile managed by the dashboard, and launch.
